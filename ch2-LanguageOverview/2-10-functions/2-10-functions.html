If a function has arguments and none are provided, an error will occur.

To prevent this, provide a default value



def function1(n = 1):
   print(n)
function1()
function1(10)

1
10


All functions return a value, so we can assign functions as a value to a variable

x = function(42)

But if a function is assigned to a variable it needs a return statement



def function1(n = 1):
   print(n)
x = function1(42)
function1(x)

Result

42
None

None is returned in python when there is no return value
in a function assigned to a variable

isPrime example

def isPrime(n):
   if n <= 1:
      return False
   for x in range(2, n):
      if n % x == 0:
         return False
      else:
         return True

n = 5
if (isPrime(n)):
   print(f"{n} is prime")
else:
   print(f"{n} is not prime")
