.format

This is an easy way of inputing variables, another way would be using

  • print(f" ${enter variable}")


x = 42
print('The number {}'.format(x))

Result

The number 42

or you can do the following for the same result

x = 42
print(f'The number {x}')


.format is a method of the string object

So, we can place a string and this method within a variable



x = 2
s = "Hello, There {}".format(x)
print(s)

Result

Hello, There 42
