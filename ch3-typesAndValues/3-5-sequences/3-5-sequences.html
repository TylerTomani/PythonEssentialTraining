Arrays, Tuples, range and Dictionaries



Array

x = [1, 2, 3, 4, 5]
x[1] = 42
for i in x:
   print("i = {}".format(i))

Tuples () - immutable

x = (1, 2, 3, 4, 5)
x[1] = 42 (Wont work)
for i in x:
   print("i = {}".format(i))


Fill variable with range

x = range(5)
for i in x:
   print("i = {}".format(i))

range and step

range goes from 5 to 50, in increments of 5

x = range(5,50,5)
for i in x:
   print(f"i: {i}", end=" ", flush=True)

Result

i: 5 i: 10 i: 15 i: 20 i: 25 i: 30 i: 35 i: 40 i: 45


Ranges are immutable

But you can alter this by putting the range in a list function

x = list(range(4))
x[1] = 99
for i in x:
   print(f"i : {i}")

Result

i: 0
i: 99
i: 2 i: 3


Dictionaries

Dicitonaries are made of key value pairs

This will only return the key

dict = ['one': 1, 'two': 2,'three': 3,'four': 4]
for i in dict:    print("i {}".format(i))

Result

i: one
i: two
i: three
i: four

To get key and value do the following

dict = ['one': 1, 'two': 2,'three': 3,'four': 4]
dict['four'] = 99
for k, v in dict.items():
   print("key: {k} value: {}".format(k,v))

Result

key: one value: 1
key: two value: 2
key: three value: 3
key: four value: 99
