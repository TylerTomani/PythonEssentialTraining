There are many string methods,
here are some common ones

  • capitalize()
  • lower()
  • upper()

You can apply this methods onto strings anywhere. Including when declaring a string

str = "Example string".upper()

The same applies to format

x = "The numbers {} {}".format(8,9)


str = "this is a string lowercase"
print(str.capitalize())

Result

This is a string

Remember, you can apply the method in the variable on the string

str = "this is a string".capitalize()
print(str)


  • upper()


str = "This WAS lowercase"
print(str.upper())

Result

THIS WAS LOWERCASE


format

You can pass values from format into strings, in order

x = "Two values {} {}".format(3,7)
print(x)

Result

Two values 3 7


OR, You can pass multiple values in format by their indexs!

x = "Two values {1} {0}".format(3,7)
print(x)

Result

Two values 7 3


adjust their position

{1:<9} will left adjust 9 spaces and {0:>9} will right adjust 9 spaces

x = "Two values {1:>9} {0:>9}".format.(3,7)
print(x)

Result

Two values 7                 3

Leading 0s

{1:<09} and {0:0>9} will add 0's

x = 'Two values "{1:>09} "{0:>09}'.format.(3,7)
print(x)

Result

Two values "700000000" "000000003"
