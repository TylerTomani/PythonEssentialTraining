7-4-keywordArgs

To access key value pairs from a dictionary you must use double asterisk **

It is conventions to label the args **kvargs


def main():
   animals(Cow="moo", Pig="oink", Cat="meow", Dog="woof")

def animals(**kvargs):
   if len(kvargs):
      for k in kvargs:
         print(f"key: {k} value: {kvargs[k]}")
   else: print("Empty")

if __name__ == '__main__': main()

result

key: Cow value: moo
key: Pig value: oink
key: Cat value: meow
key: Dog value: woof

Above, we type out the dictionary as an argument to the function, but we can assign the dict to a varaible,
then pass the variable to the function using double asterisk **.
This will produce the same result as above


def main():
   x = dict(Cow="moo", Pig="oink", Cat="meow", Dog="woof")
   animals(**x)

def animals(**kvargs):
   if len(kvargs):
      for k in kvargs:
         print(f"key: {k} value: {kvargs[k]}")
   else: print("Empty")

if __name__ == '__main__': main()
