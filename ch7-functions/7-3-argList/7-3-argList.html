You can assigne a list to a variable and pass the list into a function using a * before the variable

This will provide the same result as abve

def main():
   x = ("meow","grr","prr")
   kittens(*x)

def kitten(*args):
   if (args):
      for i in args:
         print(i)
   else: print("MEOW")
